Aviation Legacy Modernization

Valtech • Software Engineer • 2019—2021

The Challenge

The Aircraft Maintenance Steering (STC) systems were running on deprecated Java 6 frameworks, creating significant security risks and preventing the integration of modern Angular-based frontends.

The primary goal was to modernize the core backend while ensuring zero downtime for critical aviation maintenance operations. This required a meticulous migration strategy that wouldn't disrupt global maintenance schedules.

The Solution

I spearheaded the migration from Java 6 to Java 8 and reimplemented core services using Spring Boot. I also led the transition from legacy OpenLaszlo frontends to modern Angular frameworks, significantly improving the UI responsiveness for maintenance engineers.

The Impact
