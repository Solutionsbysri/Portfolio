Modernizing Payroll Infrastructure

Paychex INC • Senior Software Consultant • 2021—Present

The Challenge

The legacy system faced 20% latency issues during peak processing windows and lacked the security protocols required for modern API integrations.

As the Product Lead, I had to bridge the communication gap between stakeholders requesting "faster features" and the engineering team dealing with mounting technical debt.

The Solution

We implemented a two-phased approach. First, we migrated critical security layers to Kong Gateway. Second, we leveraged Figma to build a user-centric UI that reduced customer support tickets by 15%.
